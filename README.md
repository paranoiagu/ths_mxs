# 股票量化回测与指标查找系统——基于“抄底_梅小苏”指标

本项目旨在通过量化手段将同花顺中的“抄底_梅小苏”技术指标转化为可独立运行的 Python 代码，并基于此指标实现历史回测和自动寻找（Screening）股票市场的交易信号。

## 项目结构与文件说明

为了提升代码的可读性和项目的结构清晰度，已经将原本的文件名进行了语义化重命名优化，各文件及其作用如下：

*   **`mxs_formula.txt`**  同花顺中“抄底_梅小苏”的原始指标公式说明（源码文件）。

*   **`ths_api_manual.txt`**  同花顺公式与 API 的说明书册，作为编写及理解源公式的重要参考。

*   **`mxs_indicator.py`** “抄底_梅小苏”指标核心逻辑的 Python 实现版本，将同花顺内置的数学逻辑及各类条件判断用 Python 进行原样封装或模拟复现。

*   **`backtest.py`** 回测程序核心文件。利用 `mxs_indicator.py` 转换而成的 Python 版指标，通过带入历史股票行情数据模拟运行，以检验该公式指标在实际或历史走势下的交易收益、胜率等回测结果。

*   **`stock_screener.py`** 选股/信号提示程序文件。基于“抄底_梅小苏”指标在全市场（或自选股票池）的最新行情中寻找触及 **买入** (Buy) 或 **卖出** (Sell) 信号的个股，作为每日量化交易信号的扫描生成器。

*   **`util.py`**
    常用的基础辅助模块和工具类，例如第三方组件连接（如 Tushare 接口初始化等）、数据清洗和其他通用函数。

## 功能介绍

1.  **公式解析复刻**：将同花顺内部公式转化并构建 Python 的指标函数，确保其信号能够在量化框架中脱机运行。
2.  **交易回测引擎**：验证并复盘梅小苏抄底公式的历史胜率与收益。
3.  **交易信号扫描**：基于 Python 脚本一键寻找当日/设定时间段内触发抄底和卖出信号的特定股票标的。

## 环境依赖与安装

1. **配置 Python 环境**：建议使用 Python 3.10+。
2. **安装依赖**：项目根目录下运行如下命令安装相关库包：
   ```bash
   pip install -r requirements.txt
   ```
3. **设置环境变量**：使用本项目获取股票数据依赖 Tushare，请先在根目录创建 `.env` 文件（或重命名 `.env.example`），并填入您的 Tushare 凭证（Token）：
   ```env
   # .env 文件示例
   TUSHARE_TOKEN=your_tushare_token_here
   ```

## 使用方法

### 1. 指标测试与计算 (`mxs_indicator.py`)
独立对某只标的计算“抄底_梅小苏”的各个指标结果并导出（更贴近同花顺原显示的效果）。
```bash
python mxs_indicator.py --code 300760
```
**可用参数**：
- `--code` (必填): 6位证券代码或完整 ts_code (例如 `300760` 或 `300760.SZ`)
- `--start-date`: 计算起始日期，如 `20230101` (若不填则默认跑近一年)
- `--end-date`: 计算结束日期，如 `20231231` (若不填则默认至今日)
- `--index-code`: 强制指定对照的基准指数代码 (如 `399006.SZ`)
- `--output-csv`: 计算结果导出指定的 CSV 路径

### 2. 交易回测引擎 (`backtest.py`)
基于“抄底_梅小苏”转换而成的 Python 版指标，对个股进行买卖测算，最后输出交易记录明细、总胜率以及区间收益率等数据。
```bash
python backtest.py --code 300760 --start-date 20220101 --end-date 20240101
```
**可用参数**：与 `mxs_indicator.py` 基本一致。该脚本主要会将计算出的持仓信号和买卖点转为量化交易盈亏结果。

### 3. 多股信号扫描选股 (`stock_screener.py`)
在指定的多个日期范围内，扫描全市场或特定列表中触发“买入”或“卖出”信号且未被反向信号覆盖的股票，作为每日股票池的筛选辅助。
```bash
# 例：扫描全市场创业板最近3天内的“买入”信号，并输出到买点列表
python stock_screener.py --market 创业板 --scan-type buy

# 例：只针对特定关注股票列表寻找买卖点
python stock_screener.py --codes 300760,000001,600519 --scan-type both
```
**可用参数**：
- `--codes`: 指定扫描的一批代码，逗号分隔 (会忽视 `--market` 参数)
- `--market`: 按市场过滤范围进行全局扫描，可选 `主板`, `创业板`, `科创板`, `CDR`, `北交所`
- `--start-date` / `--end-date`: 扫描区间的开始和结束日期 YYYYMMDD
- `--scan-type`: 扫描类型，可选 `buy` (买入信号), `sell` (卖出信号), `both` (买卖信号同时扫，并分别打印)
- `--output-csv`: 输出 CSV 保存路径 (仅当 scan-type 为单向扫描时不写两者时起作用)
